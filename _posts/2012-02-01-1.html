$(function() {
     // Handler for .ready() called.
});

在jQuery文件里面有定义$=jQuery;用$("#id")和用jQuery("#id")是一样的。只是$在其它的js框架里面也被用过，可能出现重复。

var Hello = jQuery.noConflict();

这样以后就可以用Hello做前缀了

Hello("txt1");

以下三种方式相同：

1. $(document).ready(handler) 

$(document).ready(function() {
  // Handler for .ready() called.
});

2. $(handler) 

$(function() {
     // Handler for .ready() called.
});

3. $().ready(handler)（不建议使用）
