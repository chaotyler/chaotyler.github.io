### parseInt(x, y)
	y省略时：x以0x开头，16进制；x以0开头，8进制；其他10进制。
### x.toString(a)
	a是设置转换成什么进制。
	(123).toString(2) ==>  "1111011"
### ==和!!强制转换时的特例：
	undefined == null ==>  true;//其他数据与两者比较都是false
	123 == "123" ==>    true;//数字与字符串比较时能转换数字的转换成数字比较
	2e800 == Infinity ==> true;
	NaN == NaN ==> false;
	1 == 1.0 ==> true;
	!!强制转换：+-0/undefined/null/""转换成false，其他true。

			}

				margin: 0 auto;

			}
