--- layout:post title:javascript知识点之灵活限定各个文件的变量空间 ---

  1. (function() { // 匿名函数  
  2.   
  3. function log(msg) {  
  4.     console.log(msg);  
  5. }  
  6.   
  7. // 其他代码  
  8.   
  9. }()); // 立即执行 

该方式用于限定该文件内的所有变量仅局部访问，详情参靠“一等公民函数”博客；


如需在其他文件中调用该包内的元素，可将该元素直接声明为全局变量；


注意：html元素直接调用的action也必须声明为全局变量才可访问，否则会提示方法未定义。
