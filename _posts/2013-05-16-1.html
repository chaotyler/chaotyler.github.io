--- layout: post title: HTML的a标签href设为零操作的原因和方式 ---

原因

An <a> element is invalid HTML unless it has either an href or name attribute.

If you want it to render correctly as a link (ie underlined, hand pointer, etc), then it will only do so if it has a href attribute.

：一个a元素只有存在href属性或者name属性才算是有效的元素，如果你想渲染成链接的样式(如下划线和光标变化成手形)，必须添加href属性。

方式

1. href="javascript:;"

2. href="javascript:void(0);"

3. href="javascript:return false;"

4. href="javascript:event.preventDefault();"
