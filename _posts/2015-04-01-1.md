---

git init(初始化本地git库)
git clone https://github.com/chaotyler/tvplatform.git
git add .(所有改动的目录文件及子目录)/*(当前目录中的文件)/filename1 filename2(指定文件列表)
git status -s(简要描述，若不加会有更多提示内容)
git diff
git commit -m "message"(必须先用add添加到stage area才能被commit)
git commit -am "message"(集成add)
git reset HEAD -- filename1 filename2(unstage)
git rm filename(删除文件)
git branch(查看所有branch)
git branch newbranch(创建新的branch)
git checkout branchname(切换到另外的branch)
git checkout -- filename1/.(取消工作目录中的改动)
git pull(更新内容)
git push origin master(推送到服务器)
git remote(管理服务器库)

[working directory]<=reset add=>[staged area]<=revert =>commit[commit]<=pull push=>[server]
touch .gitignore或者创建一个txt文件然后编辑另存为.gitignore
